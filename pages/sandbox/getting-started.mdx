# Getting started

The Module Sandbox is built on top of [Next.js](https://nextjs.org/docs), [Viem](https://viem.sh/docs/getting-started) and [Permissionless.js](https://docs.pimlico.io/permissionless/). To get started, head to the [Sandbox repo](https://github.com/rhinestonewtf/sandbox) and create a new repo from the template.

## Installation

```bash copy
yarn install
```

## Usage

1. Create a `.env` file based on the `.env.example` file
2. Run `yarn dev` to start the development server
